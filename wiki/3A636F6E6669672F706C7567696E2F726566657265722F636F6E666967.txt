#freeze
*IGNORE

URI that doesn't display the link origin is defined.
http:// is unnecessary because it is linked when defining it here.

// リンク元の表示をさせない URI を定義します。
// ここで定義するとリンクされるため http:// は不要です。

|localhost/|

*COLOR

The header color when the list display is done is specified.
// 一覧表示する際のヘッダー色を指定します。

- ''cur'' is a current line of the line.
- ''etc'' is a line that doesn't line up.

// - ''cur'' は、整列のカレント行
// - ''etc'' は、整列されていない行

|FLAG|COLOR|h
|cur|BGCOLOR(#ff7700):|
|etc|BGCOLOR(#cccccc):|
